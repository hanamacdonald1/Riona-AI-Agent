## 🔧 **API Endpoints Available**

### 🔓 **Public Endpoints**
```
GET  /api/status              - System health check
GET  /api/system/health       - Detailed system information
GET  /api/agent/characters    - Available AI characters
```

### 🔐 **Authenticated Endpoints**
```
POST /api/login               - User authentication
GET  /api/me                  - Check auth status
POST /api/logout              - User logout

POST /api/interact            - Start Instagram automation
POST /api/dm                  - Send direct messages
POST /api/scrape-followers    - Scrape follower lists
DELETE /api/clear-cookies     - Clear Instagram session

POST /api/agent/generate      - Generate AI content
POST /api/agent/train/youtube - Train with YouTube
POST /api/agent/train/website - Train with websites
POST /api/agent/train/file    - Train with file uploads

GET  /api/analytics           - Performance analytics
GET  /api/system/logs         - Activity logs
```
